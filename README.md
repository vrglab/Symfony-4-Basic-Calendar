# Symfony 4 Basic Calendar
So basically I was bored and wanted to do something with Symfony4 So I made this calendar.

It handles most things in regards to showing todays date and exploring other dates.

## On the Agenda
1. Taking notes in Days
2. German translations
3. Switching from Dark to light mode (Frontend only, since this is supported already)

## Used Tech
1. php 7.1
1. Symfony4
2. Twig
3. Webpack encore
4. Doctrine
5. TailwindCSS
6. PostCSS
7. Mysql
8. Composer + NPM
